## 核心能做什么？

不说废话，直接列几个最实用的能力：

**访问权限控制**：可以隐藏开始菜单、禁用任务栏、锁定桌面，甚至给特定文件夹设置独立密码。你不想让别人碰的地方，就真的碰不到。

**阻止软件和进程运行**：不想让人装软件、开游戏、点进下载器？直接拉黑就好，拦截特定程序和弹窗。

**限时访问**：设定使用时限，超过时间自动关机。家里给孩子的电脑用这个非常顺手，不用盯着。

**多用户独立配置**：针对不同用户账号单独设置权限，张三能用什么、李四看不到什么，各管各的。

**日志监控和邮件通知**：用户做了什么操作，全部写入日志，甚至可以发邮件实时通知你。

**展台模式（Kiosk Mode）**：适合 ATM 机、自助终端等公共设备，强制锁定在特定程序里。

**家长**：给孩子的电脑设置上网时间上限、屏蔽不良网站、锁住系统设置，不用每天追着管。专业版的网站内容过滤器很好用。

**中小企业 / 办公室管理员**：公共电脑、前台机器、员工共用工位——不想让人随便改设置、装软件、访问公司敏感目录，标准版就够了，专业版更稳。
